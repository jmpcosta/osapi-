Detailed Description

An implementation of a configuration parser interface that reads a set of CIs.

A file parser imports Configuration Items into a configuration. Each line is in the form:

[container name] | [Configuration Item type]

Where a Configuration type can be:

prop:[property name]=[property value]
env:[variable name]=[variable value]
arg:[arguments]
